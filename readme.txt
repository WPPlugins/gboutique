== Description ==

Simple e-commerce for wordpress using google spreadsheet and paypal


== Installation ==

1. Send `gboutique` to the folder `/wp-content/plugins/`
2. Activate le plugin through 'Extensions' in WordPress Admin Panel
3. Go to 'Settings -> gboutique'
4. More help on : http://wpcb.fr/gboutique (en Français)
